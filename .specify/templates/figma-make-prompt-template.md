## Follow-up Prompts

Use these to iterate on the generated design:

### Style Adjustments
- "Use a darker color scheme"
- "Make it more playful with rounded corners"
- "Apply our brand colors: primary #XXXXXX, secondary #XXXXXX"
- "Use [Font Name] for typography"

### Layout Changes
- "Move the [element] to the [position]"
- "Make the [component] larger/smaller"
- "Add more whitespace between sections"
- "Stack elements vertically on mobile"

### Add States
- "Add an empty state when no data"
- "Show a loading skeleton"
- "Add error state with retry button"
- "Show success confirmation modal"

### Add Interactions
- "Add swipe to delete gesture"
- "Make the card expandable on tap"
- "Add pull-to-refresh animation"
- "Show bottom sheet on button tap"

## How to Use

1. Open [Figma Make](https://www.figma.com/make)
2. Start a new project or open existing project
3. Paste the main prompt above
4. Review generated design
5. Use follow-up prompts to iterate
6. Export or continue editing in Figma
